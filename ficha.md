## 3. Houve comando de build?

Não. O projeto é composto apenas por arquivos HTML, CSS e JavaScript.

## 4. Diferença entre abrir o arquivo localmente e acessar pela URL pública

Localmente, a página funciona apenas no computador onde os arquivos estão armazenados. Pela URL pública, ela fica disponível na Internet e pode ser acessada por qualquer pessoa.
