# Prática — Hospedagem de página estática

## Objetivo
Disponibilizar uma página simples na Internet usando uma hospedagem estática.

## O que pode ser alterado
Você pode alterar os textos de `index.html`, `styles.css` e `script.js`.

## Entrega
Registre:

1. URL pública gerada pela plataforma;
2. quais arquivos foram hospedados;
3. se houve comando de build;
4. diferença entre abrir o arquivo localmente e acessar pela URL pública.

## Observação
Este projeto não possui back-end, Node.js em execução nem banco de dados.
